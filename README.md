# 📊 Sales Performance Dashboard (Looker Studio)

## Overview

This project presents an interactive sales dashboard developed in **Looker Studio** to monitor commercial performance across customers, countries, and time periods.

The dashboard enables users to analyze sales trends, compare performance by country, evaluate customer contributions, and monitor key business metrics through an intuitive interface.

---

## Business Problem

Organizations require a centralized reporting solution to monitor sales performance and identify trends that support commercial decision-making.

This dashboard provides a clear overview of:

- Total Sales
- Units Sold
- Sales Trends
- Sales by Country
- Sales by Customer
- Budget Performance
